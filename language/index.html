What is Ux and UI

Client Side Web Development Intro

UX:

In web design UX stands for user experience, and is used to describe how a user might interact with the service. As well as how the service will be used by the user, is it efficient, and is it easy to use.

Important Characteristics of UX:

  • Usability
  • Ability to meet customer needs
  • Level of simplicity: Avoid overly complex, simply elegant is the goal
  • Number of disciplines: marketing, design, etc.
  • The 5 questions : Who, What, Where, When, Why the product will be used
For more Infomation on Ux Design

UI:

On the other hand UI in web desin is the user interface. This focuses more on the perception of the product. The overall asthetic amd behavior of the product

Important Characteristics of UI:

  • Navigation: slide bars, back arrows, links to other pages, search fields, etc.
  • Interactivity: How does the interface work and behave
  • Visual Desgin: What does it look like, color, fonts, images, icons, logos, etc.
  • Input Controls: buttons, text fields, checkboxes, etc.
  • Organization: How is the information and other elements formated on the interface
  • Visual Hierachy: How are the different elements emphasized, color contrasts, shape, size, spacing, etc.
